About So Long Map Maker

So Long Map Maker is a free and open-source map maker for the game 42School So Long Project. Create your own maps and share them with the community! This project was created by INovomiast2 (ivnovomi) as a part of the 42School curriculum.

Features

  • Map editor with drag and drop functionality
  • Save and load maps
  • Share maps with the community
  • Rate and comment maps
  • Report inappropriate maps

Technologies

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python, MySQL
  • Deployment: GitHub Pages

Source Code

The source code for this project is available on GitHub.

Feedback

If you have any feedback or suggestions for this project, please feel free to contact me on 42Intra or Comunity Feedback.
